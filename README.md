# Pasaj Clone

TypeScript ile hazırlanmış 1. hafta staj projesi iskeleti.

## Kurulum

```bash
npm install
npm run dev
```

## VS Code önerilen eklentiler

- ESLint
- Prettier
- TypeScript and JavaScript Language Features
- Code Spell Checker
- Auto Rename Tag
- Error Lens

`.vscode/extensions.json` dosyası açıldığında bu önerileri otomatik gösterir.
